# IntelliDesk AI

IntelliDesk AI is a production-ready, industry-level SaaS AI Workspace platform. It consolidates multiple productivity tools—including AI Resume Builder, Cover Letter Generator, Email Writer, AI PDF Chat (RAG), Meeting Notes Summarizer, AI Interview Coach, Image Caption Generator, and a SaaS Subscription & Admin system—into a unified dashboard.

## Tech Stack
- **Frontend**: React (Vite + TypeScript), Tailwind CSS, Shadcn UI, React Router v6, TanStack Query, Framer Motion
- **Backend**: Node.js + Express + TypeScript, Prisma ORM
- **Database**: PostgreSQL
- **Payments**: Stripe

## Project Structure
- `client/`: React Web Application
- `server/`: Express backend API & Prisma models
